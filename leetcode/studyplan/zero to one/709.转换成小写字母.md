# LeetCode 709 - 转换成小写字母
`https://leetcode.cn/problems/to-lower-case/description/?envType=study-plan-v2&envId=programming-skills`

给你一个字符串 s ，将该字符串中的大写字母转换成相同的小写字母，返回新的字符串。

## 思路

遍历，判断ascll码范围，转换，输出。

### 问题

- 1.可以用'A'和'Z'代替数字，更易读且不用记忆数字。
- 2.可以直接s[i]+=32,不用创建int ascii。

### 时/空间复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 最终代码

```cpp
class Solution {
public:
    string toLowerCase(string s) {
        int sz=s.size();
        for (int i=0;i<sz;i++){
            char ascii= s[i];
            if (ascii>='A'&&ascii<='Z') s[i]=ascii+32;
        }
        return s;
    }
};
```

### 改进
- 1.用A和Z代替了ascii编码
- 2.ascii代替s[i]其实无关紧要，性能差不多，主要是代码风格的区别。

### 时/空间复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
