# LeetCode 1768 - 交替合并字符表

`https://leetcode.cn/problems/merge-strings-alternately/description/?envType=study-plan-v2&envId=programming-skills`

给定两个 字符串 word1和word2。从 word1 开始，通过交替添加字母来合并字符串。如果一个字符串比另一个字符串长，就将多出来的字母追加到合并后字符串的末尾。
返回 合并后的字符串 。

## 思路

`leetcode做的第一道题，先用python解了一遍.
而后开始学习使用c语言。`

创建一个新字符串，遍历两个旧字符串，创建两个临时变量记录循环次数，并在循环中与字符串长度比较。合并更长的字符串剩余部分后返回字符串。

## 标签

- 字符串
- 双指针（简单应用）

## 初始代码

```cpp
class Solution {
public:
    string mergeAlternately(string word1, string word2) {
        string ans;
        int n=0;
        //以下
        for(int i=0;i<word1.length()&&i<word2.length();i++){
        //
            ans+=word1[i];
            ans+=word2[i];
            n+=1;
        }
        while (n<word1.length()){
            ans+=word1[n];
            n+=1;
        }
        while (n<word2.length()){
            ans+=word2[n];
            n+=1;
        }
        return ans;
    }
};
```
### 问题

- 1.更多的临时变量
- 2.重复的length计算

### 时/空间复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 最终代码

```cpp
class Solution {
public:
    string mergeAlternately(string word1, string word2) {
        string ans;
        int n=0;
        //以下
        int len1=word1.length();int len2=word2.length();
        //
        while(n<len1&&n<len2){
        //
            ans+=word1[n];
            ans+=word2[n];
            n+=1;
        }
        while (n<len1){
            ans+=word1[n];
            n+=1;
        }
        while (n<len2){
            ans+=word2[n];
            n+=1;
        }
        return ans;
    }
};
```

### 改进
- 1.减少临时变量数量
- 2.将重复计算的length换成常量

### 时/空间复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 收获

- 1.leetcode 题目一般是return答案而非print。
- 2.在可能的情况下不要创建过多的变量。
- 3.将一些函数计算后的结果存储到常量中，减少消耗。
