# LeetCode 459 - 重复的子字符串
`https://leetcode.cn/problems/repeated-substring-pattern/description/?envType=study-plan-v2&envId=programming-skills`

给定一个非空字符串s，检查是否可以由子字符串重复多次构成

## 思路

将子字符串长度由1逐渐增加，直至长度=s长度的一半为止。验证该子字符串是否可以重复构成母字符串。

## 标签

- 字符串匹配
- 字符串

## 初始代码

```cpp
class Solution {
public:
    bool repeatedSubstringPattern(string s) {
        int lens=s.length();int same=0;
        int start=same+1;
        while (same<=(lens/2)&&start+same+1<=lens){
            while (s.substr(0,same+1)==s.substr(start,same+1)){
                if ((start+same+1)==lens) return true;
                start+=same+1;
            }
            same+=1;
            start=same+1;
        }
        return false;
    }
};
```

## 最终代码

### A （substr优化版a）

```cpp
class Solution {
public:
    bool repeatedSubstringPattern(string s) {
        int lens=s.length();int lens2=lens/2;int lens3=sqrt(lens);
        vector<int> nums;
        int same=0;
        int start=0;
        //lens/(1--lens3)
        for (int i=1;i<=lens3;i++){
            if (lens%i==0){
                nums.push_back(i);
                nums.push_back(lens/i);
            }
        }
        nums.erase(remove(nums.begin(),nums.end(),lens),nums.end());
        for (int i:nums){
            start= i;
            while (s.substr(0,i)==s.substr(start,i)){
                if ((start+i)==lens) return true;
                start+=i;
            }
        }
        return false;
    }
};
```

### 改进
- 1.减少了枚举的次数，通过提前计算因数和其他条件减少substr的次数。
- 2.执行时间和消耗内存大幅降低。

### 时/空间复杂度

- 时间复杂度：O(n^2)
- 空间复杂度：O(n)

### B （讨论区--substr优化版b--时间优化）

```cpp
class Solution {
public:
    bool repeatedSubstringPattern(string s) {
        string t = s + s;
        return t.substr(1, t.size() - 2).find(s) != string::npos;
    }
};
```

### 改进
- 1.通过巧妙的掐头去尾验证法，减少运行次数，甚至一个字符串只需要计算一次。最大幅度的降低了运行时间。

### C （gpt-空间优化版）

```cpp
class Solution {
public:
    bool repeatedSubstringPattern(string s) {
        int n = s.size();

        for (int len = 1; len <= n / 2; ++len) {
            if (n % len != 0) continue;   // 长度必须能整除 n

            bool ok = true;
            for (int i = len; i < n; ++i) {
                if (s[i] != s[i % len]) {
                    ok = false;
                    break;
                }
            }

            if (ok) return true;
        }

        return false;
    }
};
```

### 改进
- 1.改substr为创造新int，减少空间复杂度，从而达到消耗内存最低。
- 2.

### 时/空间复杂度

- 时间复杂度：O(n^2)
- 空间复杂度：O(1)
