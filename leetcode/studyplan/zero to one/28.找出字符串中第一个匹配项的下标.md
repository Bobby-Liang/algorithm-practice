# LeetCode 28 - 找出字符串中第一个匹配项的下标
`地址`

给定两个字符串h、n。找到h中和n字符串第一个匹配项的下标，如果不是一部分，则返回-1

## 思路

- I. 遍历h，并且在遍历过程中与n的第一个字符进行比较，并验证n是否存在于h中。

## 标签

- 双指针
- 字符串

## 初始代码

```cpp
class Solution {
public:
    int strStr(string haystack, string needle) {
        int pointerA = 0;int pointerB = 0;int finish=0;int lengthH=haystack.length();int lengthN=needle.length();
        int ans=-1;
        while (lengthN!=0&&lengthH!=0&&true){
            pointerB=ans+1;
            ans= -1;
            while (pointerA<lengthN&&pointerB<lengthH&&needle[pointerA]!=haystack[pointerB]){
                pointerB++;
                if (pointerB==lengthH){
                    return -1;
                }
            }
            ans = pointerB;
            if (pointerB+lengthN>lengthH) return -1;
            while(pointerA<lengthN&&pointerB<lengthH&&needle[pointerA]==haystack[pointerB]&&pointerB+lengthN - pointerA<=lengthH){
                pointerA++;
                pointerB++;
                if (pointerA==lengthN){
                    return ans;
                }
            }
            pointerA=0;
        }
        return ans;
    }
};
```
### 问题

- 1.变量命名没有特点，看似不是int a/string b，实际上导致使用过程中发生混乱、错误。
- 2.条件判断顺序不当，条件繁杂。
- 3.未使用的变量未删除。
