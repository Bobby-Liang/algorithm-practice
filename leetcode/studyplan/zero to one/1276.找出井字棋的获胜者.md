# LeetCode 1276 - 找出井字棋的获胜者
`https://leetcode.cn/problems/find-winner-on-a-tic-tac-toe-game/description/?envType=study-plan-v2&envId=programming-skills`

井字棋 是由两个玩家 A 和 B 在 3 x 3 的棋盘上进行的游戏。井字棋游戏的规则如下：

玩家轮流将棋子放在空方格 (' ') 上。
第一个玩家 A 总是用 'X' 作为棋子，而第二个玩家 B 总是用 'O' 作为棋子。
'X' 和 'O' 只能放在空方格中，而不能放在已经被占用的方格上。
只要有 3 个相同的（非空）棋子排成一条直线（行、列、对角线）时，游戏结束。
如果所有方块都放满棋子（不为空），游戏也会结束。
游戏结束后，棋子无法再进行任何移动。
给你一个数组 moves，其中 moves[i] = [rowi, coli] 表示第 i 次移动在 grid[rowi][coli]。如果游戏存在获胜者（A 或 B），就返回该游戏的获胜者；如果游戏以平局结束，则返回 "Draw"；如果仍会有行动（游戏未结束），则返回 "Pending"。

你可以假设 moves 都 有效（遵循 井字棋 规则），网格最初是空的，A 将先行动。

## 思路

a.创建两个数组，分别存放玩家a和b的下棋位置，再分别遍历两个数组，如果有谁的下棋位置符合“胜利棋盘”的样子，则返回谁赢了。若谁都没赢，且下满了9步，就返回Draw，没下满返回Pending。

但是太复杂了，故未实现。

b.创建两个数组，分别记录每一行、每一列的落子情况，再创建两个整数，记录主副对角线的落子情况。先遍历数组，a在某行某列下了棋，那么数组中代表这行和这列的常数就+=1，b则-=1，同时判断是否在对角线上，若在则与行列相同，a+=1，b-=1。

## 初始代码

```cpp
class Solution {
public:
    string tictactoe(vector<vector<int>>& moves) {
        int row[3] = {0};int column[3] = {0};int maindiagonal=0;int antidiagonal=0;
        int n=0;
        int sz= moves.size();
        if (sz<5) return "Pending";
        for (auto& v:moves){
            if (n==0||n%2==0) {
                row[v[0]]+=1;column[v[1]]+=1;
                if (v==vector<int>{0,0}||v==vector<int>{1,1}||v==vector<int>{2,2}){
                    maindiagonal+=1;
                }
                if (v==vector<int>{0,2}||v==vector<int>{1,1}||v==vector<int>{2,0}){
                    antidiagonal+=1;
                }
                n+=1;
            }
            else {
                row[v[0]]-=1;column[v[1]]-=1;
                if (v==vector<int>{0,0}||v==vector<int>{1,1}||v==vector<int>{2,2}){
                    maindiagonal-=1;
                }
                if (v==vector<int>{0,2}||v==vector<int>{1,1}||v==vector<int>{2,0}){
                    antidiagonal-=1;
                }
                n+=1;
            }
        }
        if (row[0]==3||row[1]==3||row[2]==3||column[0]==3||column[1]==3||column[2]==3||maindiagonal==3||antidiagonal==3) return "A";
        if (row[0]==-3||row[1]==-3||row[2]==-3||column[0]==-3||column[1]==-3||column[2]==-3||maindiagonal==-3||antidiagonal==-3) return "B";
        if (n==9) return "Draw";
        return "Pending";
    }
};
```

## 最终代码

```cpp
class Solution {
public:
    string tictactoe(vector<vector<int>>& moves) {
        int row[3] = {0};int column[3] = {0};int maindiagonal=0;int antidiagonal=0;
        int n=0;
        int sz= moves.size();
        if (sz<5) return "Pending";
        for (auto& v:moves){
            if (n%2==0) {
                row[v[0]]+=1;column[v[1]]+=1;
                if (row[v[0]]==3||column[v[1]]==3) return"A";
                if (v[0]==v[1]){
                    maindiagonal+=1;
                    if (maindiagonal==3) return "A";
                }
                if (v[0]+v[1]==2){
                    antidiagonal+=1;
                    if(antidiagonal==3) return "A";
                }
                n+=1;
            }
            else {
                row[v[0]]-=1;column[v[1]]-=1;
                if (row[v[0]]==-3||column[v[1]]==-3) return"B";
                if (v[0]==v[1]){
                    maindiagonal-=1;
                    if(maindiagonal==-3) return "B";
                }
                if (v[0]+v[1]==2){
                    antidiagonal-=1;
                    if(antidiagonal==-3) return "B";
                }
                n+=1;
            }
        }
        if (n==9) return "Draw";
        return "Pending";
    }
};
```
