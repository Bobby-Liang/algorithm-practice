# LeetCode 67 - 二进制求和
`https://leetcode.cn/problems/add-binary/?envType=study-plan-v2&envId=programming-skills`

给你两个二进制字符串 a 和 b ，以二进制字符串的形式返回它们的和。

## 思路

因为是二进制求和，字符串只会有0,1两种状态。把a,b中较短的一个按位加到a上，最后遍历一遍a，当前一位是0/1则不变，如果是2/3则当前位为0/1,后一位+=1。最后一位单独判断，若为2/3则在字符串最前方多加一位"1"。返回。

## 初始代码

```cpp
class Solution {
public:
    string addBinary(string a, string b) {
        int sza=a.size();int szb=b.size();int aa=sza-1;int bb=szb-1;

        if (sza>szb){
            while (bb>=0){
                if (b[bb]=='1'){
                    if (a[aa]=='1') a[aa]='2';
                    else if (a[aa]=='0') a[aa]='1';
                }
                else {
                }
                bb-=1;
                aa-=1;
            }
            aa=sza-1;
            while(aa>0){
                if (a[aa]=='2') {
                    if(a[aa-1]=='1') a[aa-1]='2';
                    else if(a[aa-1]=='2') a[aa-1]='3';
                    else if(a[aa-1]=='0') a[aa-1]='1';
                    a[aa]='0';
                }
                else if (a[aa]=='3') {
                    if(a[aa-1]=='1') a[aa-1]='2';
                    else if(a[aa-1]=='2') a[aa-1]='3';
                    else if(a[aa-1]=='0') a[aa-1]='1';
                    a[aa]='1';
                }
                aa-=1;
            }
            if (a[0]=='2'){
                a[0]='0';
                a="1"+a;
            }
            else if(a[0]=='3'){
                a[0]='1';
                a="1"+a;
            }
            return a;
        }
        else {
            while (aa>=0){
                if (a[aa]=='1'){
                    if (b[bb]=='1') b[bb]='2';
                    else if (b[bb]=='0') b[bb]='1';
                }
                else {
                }
                bb-=1;
                aa-=1;
            }
            bb=szb-1;
            while(bb>0){
                if (b[bb]=='2') {
                    if(b[bb-1]=='1') b[bb-1]='2';
                    else if(b[bb-1]=='2') b[bb-1]='3';
                    else if(b[bb-1]=='0') b[bb-1]='1';
                    b[bb]='0';
                }
                else if (b[bb]=='3') {
                    if(b[bb-1]=='1') b[bb-1]='2';
                    else if(b[bb-1]=='2') b[bb-1]='3';
                    else if(b[bb-1]=='0') b[bb-1]='1';
                    b[bb]='1';
                }
                bb-=1;
            }
            if (b[0]=='2'){
                b[0]='0';
                b="1"+b;
            }
            else if(b[0]=='3'){
                b[0]='1';
                b="1"+b;
            }
            return b;
        }
        return "0";
    }
};
```
