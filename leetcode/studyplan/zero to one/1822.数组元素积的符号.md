# LeetCode 1822 - 数组元素积的符号
`https://leetcode.cn/problems/sign-of-the-product-of-an-array/?envType=study-plan-v2&envId=programming-skills`

一个数组，有正数有负数有0。如果相乘到最后小于0返回-1，大于0返回1，等于0返回0。

## 思路

设定一个n=1，每个数和0比较，如果这个数<0则n*-1，>0不变，等于0则直接返回0。

## 初始代码

```cpp
class Solution {
public:
    int arraySign(vector<int>& nums) {
        int n=1;
        for(int i:nums){
            if (i<0){
                n*=(-1);
            }
            else{
                if (i==0){
                    return 0;
                }
            }
        }
        return n;
    }
};
```
