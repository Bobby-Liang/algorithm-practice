# LeetCode 1523 - 在区间范围内统计奇数数目
`https://leetcode.cn/problems/count-odd-numbers-in-an-interval-range/description/?envType=study-plan-v2&envId=programming-skills`

给你两个非负整数 low 和 high 。请你返回 low 和 high 之间（包括二者）奇数的数目。

## 思路

a.for遍历low到hihgh，计数。
b.根据low和high的奇偶性直接计算这个区间内的奇数个数。

## 初始代码

```cpp
class Solution {
public:
    int countOdds(int low, int high) {
        int ans=0;
        for(int i =low;i<=high;i++){
            if (i%2!=0) {
                ans+=1;
            }
        }
        return ans;
    }
};
```

## 最终代码

```cpp
class Solution {
public:
    int countOdds(int low, int high) {
        int n=high-low;
        if (high==low){
            if (high%2==0) return 0;
            return 1;
        }
        if (high%2==0&&low%2==0) return (high-low)/2;
        if ((high%2==0&&low%2!=0)||(high%2!=0&&low%2==0)) return ((high-low)+1)/2;
        return (high-low)/2+1;
    }
};
```
