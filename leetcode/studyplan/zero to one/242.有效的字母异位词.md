# LeetCode 242 - 有效的字母异位词
`https://leetcode.cn/problems/valid-anagram/?envType=study-plan-v2&envId=programming-skills`

给定两个字符串，判断他们是否是字母异位词。是返回true，否返回false。

## 思路

- 哈希图统计出现次数，相减后如果值均为0则true，反则为false。
- 这道题同样可以使用sort函数解决。

## 标签

- 哈希图
- 字符串

## 初始代码

### A

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        unordered_map<char,int> hashs;
        for (char a:s){
            hashs[a]++;
        }
        for (char a:t){
            hashs[a]--;
        }
        for (auto a:hashs){
            if(a.second!=0) return false;
        }
        return true;
    }
};
```

### B

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        if (s.length()!=t.length()) return false;
        sort(s.begin(),s.end());
        sort(t.begin(),t.end());
        return s==t;
    }
};
```
