# LeetCode 68 - 加一
`地址`

给定一个vector数组，每个数字i：0<=i<=9。在数组末尾+1，如[1,3,8]-->[1,3,9];[9,9,9]-->[1,0,0,0]。
返回数组

## 思路

从后向前遍历数组，若着这一位为9，则变为0，向前遍历；如果不是9，则+1输出。若均为9，则在最前方加一位1，输出。

## 初始代码

```cpp
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        int num =0;int sz =digits.size()-1;int n=0;
        for(int i:digits){
            num+=i*pow(10,sz);
            sz-=1;
        }
        num +=1;
        string numstr =to_string(num);
        int sza=numstr.size();
        if (sz !=sza) digits.resize(sza);
        while (n < sza){
            digits[n] =numstr[n]-'0';
            n+=1;
        }
        return digits;
    }
};
```

## 最终代码

```cpp
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        for(int i=(int)digits.size() - 1;i>=0;i--){
            if (digits[i]<9){
                digits[i]+=1;
                return digits;
            }
            else{
                digits[i]=0;
            }
        }
        digits.resize(digits.size()+1);
        digits[0]=1;
        return digits;
    }
};
```
