# LeetCode 682 - 棒球比赛
`地址`

给定一个字符串动态数组，其中包含数字、+、C、D这些字符串元素。数字表示本回合得分，C表示上次得分无效，D表示本轮分数是上个分数的二倍，+表示本轮分数是前两次分数和。求总分，返回。

## 思路

遍历字符串数组，判断每个元素是什么。创建一个动态数组作栈，每次的得分入栈，无效就出栈，并记录每次变化后的总分，返回总分。

## 标签

- 栈
- 数组

## 初始代码

```cpp
class Solution {
public:
    int calPoints(vector<string>& operations) {
        vector<int> ansv;
        int ans=0;
        int n=-1;
        int ints =0;
        int d=0;
        int ad = 0;
        for(string s:operations){
            if (s=="+"){
                ad = ansv[n]+ansv[n-1];
                ans += ad;
                ansv.push_back(ad);
                n +=1;
                
            }
            else if (s=="D"){
                d=(ansv[n]*2);
                ans +=(d);
                ansv.push_back(d);
                n+=1;
            }
            else if (s=="C"){
                ans -= ansv[n];
                ansv.pop_back();
                n-=1;
            }
            else if (isdigit(s[0]) || s[0]=='-'){
                ints = stoi(s);
                ans += ints;
                n+=1;
                ansv.push_back(ints);
            }
        }
        return ans;
    }
};
```
