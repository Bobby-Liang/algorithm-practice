# LeetCode 389 - 找不同
`https://leetcode.cn/problems/find-the-difference/?envType=study-plan-v2&envId=programming-skills`

给定两个 仅包含小写字母的 字符串s、t。t是由 打乱之后的s 再在随机位置添加一个字母所组成的。找出这个字母

## 思路

- I. 遍历字符串s，将每个字符都在t中查找并删除。直至s遍历完毕/当前字符无法在t中查找到为止。返回字符。
- II. 利用 异或 将s、t中的所有字符进行异或。
- III. 利用 int(char)的方法把s、t的所有字符转换为ascll码值并相加，而后相减再转为char。
- IV. 使用Hash Table解题。

## 标签

- 哈希表
- 字符串

## 初始代码

```cpp
class Solution {
public:
    char findTheDifference(string s, string t) {
        int lens=s.length();int lent=t.length();
        int ints=0;int intt=0;
        while (lent>intt){
            if (s.find(t[intt])==string::npos){
                break;
            }
            else{
                s.erase(s.find(t[intt]),1);
            }
            intt++;
        }
        return t[intt];
    }
};
```

## 最终代码

### A

```cpp
class Solution {
public:
    char findTheDifference(string s, string t) {
        char ans=0;
        for (char c:s) ans^=c;
        for (char c:t) ans^=c;
        return ans;
    }
};
```

### B

```cpp
class Solution {
public:
    char findTheDifference(string s, string t) {
        int a=0;int b=0;
        for(char c : s) a+=int(c);
        for(char c : t) b+=int(c);
        int d = b-a;
        return char(d);
    }
};
```

### C(Hash Tabel)

```cpp
class Solution {
public:
    char findTheDifference(string s, string t) {
        unordered_map<char,int> dict1;
        for (char c:s){
            dict1[c]++;
        }
        for (char c:t){
            dict1[c]--;
            
        }
        for (auto p:dict1){
            if(p.second!=0){
                return p.first;
            }
        }
        return 0;
    }
};
```

### 改进
- 1. 时间复杂度更低。
- 2. 代码更简洁明要。
- 3. 在C中首次使用了Hash Table.

### 时/空间复杂度

- 时间复杂度：A/B/C均为：O(n)
- 空间复杂度：A/B/C均为：O(1)
