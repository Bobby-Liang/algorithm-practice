# LeetCode 1041 - 困于环中的机器人
`https://leetcode.cn/problems/robot-bounded-in-circle/description/?envType=study-plan-v2&envId=programming-skills`

在无限的平面上，机器人最初位于 (0, 0) 处，面朝北方。注意:

北方向 是y轴的正方向。
南方向 是y轴的负方向。
东方向 是x轴的正方向。
西方向 是x轴的负方向。
机器人可以接受下列三条指令之一：

"G"：直走 1 个单位
"L"：左转 90 度
"R"：右转 90 度
机器人按顺序执行指令 instructions，并一直重复它们。

只有在平面中存在环使得机器人永远无法离开时，返回 true。否则，返回 false。

## 思路

a.执行一次完整指令后。如果回到了原点，则必定循环。要么就是执行两次后回到了原点。要么就是四次后，因为只有四个方向。如果1、2、4次后都没回到原点，那么就说明不可能回到了。

b.(gpt)执行一次完整指令后。如果回到原点 或者没回到，但是方向变了，那么就能回到原点。否则不能。

## 标签

- 数学
- 字符串

## 初始代码

```cpp
class Solution {
public:
    bool isRobotBounded(string instructions) {
        int dy=1;int dx=0;
        int y= 0;int x= 0;
        int n=0;
        while (n<4){
            for (char c:instructions){
                if (c=='R'){
                    if(dy==1&&dx==0){
                        dy-=1;dx+=1;
                    }
                    else if (dy==-1&&dx==0){
                        dy+=1;dx-=1;
                    }
                    else if (dy==0&&dx==1){
                        dy-=1;dx-=1;
                    }
                    else if (dy==0&&dx==-1){
                        dy+=1;dx+=1;
                    }
                }
                else if (c=='L'){
                    if(dy==1&&dx==0){
                        dy-=1;dx-=1;
                    }
                    else if (dy==-1&&dx==0){
                        dy+=1;dx+=1;
                    }
                    else if (dy==0&&dx==1){
                        dy+=1;dx-=1;
                    }
                    else if (dy==0&&dx==-1){
                        dy-=1;dx+=1;
                    }
                }
                else if (c=='G'){
                    y+=dy;x+=dx;
                }
            }
            n+=1;
            if (x==0&&y==0) return true;
        }
        return false;
    }
};
```
### 问题

- 1.循环了最多4次，虽然确实模拟了机器人运动路径，但是不是最优解。常数偏大。
- 2.转向方向判断可以优化。

### 时/空间复杂度

- 时间复杂度：O(n)  注：O(4n)循环了四次，偏大，可以改进。
- 空间复杂度：O(1)

## 最终代码

```cpp
class Solution {
public:
    bool isRobotBounded(string instructions) {
        int dy=1;int dx=0;
        int y= 0;int x= 0;
        for (char c:instructions){
            if (c=='R'){
                int temp =dx;
                dx=dy;
                dy=-temp;
            }
            else if (c=='L'){
                int temp =dx;
                dx=-dy;
                dy=temp;
            }
            else if (c=='G'){
                y+=dy;x+=dx;
            }
        }
        return ((x==0&&y==0)||(dy!=1||dx!=0));
    }
};
```

### 改进
- 1.优化了转向判断。
- 2.优化了循环次数，改进了判断条件。

### 时/空间复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
