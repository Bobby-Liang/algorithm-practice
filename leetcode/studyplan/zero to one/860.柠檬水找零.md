# LeetCode 860 - 柠檬水找零
`https://leetcode.cn/problems/lemonade-change/description/?envType=study-plan-v2&envId=programming-skills`

在柠檬水摊上，每一杯柠檬水的售价为 5 美元。顾客排队购买你的产品，（按账单 bills 支付的顺序）一次购买一杯。
每位顾客只买一杯柠檬水，然后向你付 5 美元、10 美元或 20 美元。你必须给每个顾客正确找零，也就是说净交易是每位顾客向你支付 5 美元。
注意，一开始你手头没有任何零钱。
给你一个整数数组 bills ，其中 bills[i] 是第 i 位顾客付的账。如果你能给每位顾客正确找零，返回 true ，否则返回 false 。

 

## 思路

a.设置三个int变量5、10、20，从头开始遍历，5元则5+1，10元则10+1、5-1，如果5不足则返回false。20元则先10-1、5-1，如果10不够就尝试5-3，如果5也不够就返回false；如果做到最后也没出问题就返回true。

b.不用设置20的这个变量，不涉及找钱。

## 标签

- 贪心算法
- 数组

## 初始代码

```cpp
class Solution {
public:
    bool lemonadeChange(vector<int>& bills) {
        int five=0;int ten=0;int twe=0;
        for (int i:bills) {
            if (i==5) five+=1;
            else if(i==10) {
                if (five>=1) {
                    ten+=1;
                    five-=1;
                }
                else return false;
            }
            else if(i==20){
                if (ten>=1&&five>=1){
                    twe+=1;
                    ten-=1;
                    five-=1;
                }
                else if(five>=3){
                    twe+=1;
                    five-=3;
                }
                else return false;
            }
        }
        return true;
    }
};
```
