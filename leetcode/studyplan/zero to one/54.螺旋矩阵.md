# LeetCode 54 - 螺旋矩阵
`https://leetcode.cn/problems/spiral-matrix/?envType=study-plan-v2&envId=programming-skills`

给你一个 m 行 n 列的矩阵 matrix ，请按照 顺时针螺旋顺序 ，返回矩阵中的所有元素。

## 思路

先遍历vector[0]中的元素，并记录在新创建的一个数组中。再保持遍历完之后的vector[i][j]的j不变，i++。再保持i不变，j--。在保持j不变，i--。在这个过程中把提前设定好的界限常数的大小进行更改。每次-1或+1，达到目标。

## 初始代码

```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int> ans;
        int i=0;int j=0;int n=0;int szi=matrix.size();int szj=matrix[0].size();int iO=1;int jO=0;int nmax=(szi*szj);
        while (n<nmax){
            while(j<szj){
                ans.push_back(matrix[i][j]);
                j+=1;
                n+=1;
            }szj-=1;i+=1;j-=1;if (n==nmax) return ans;
            while(i<szi){
                ans.push_back(matrix[i][j]);
                i+=1;
                n+=1;
            }szi-=1;i-=1;j-=1;if (n==nmax) return ans;
            while(j>=jO){
                ans.push_back(matrix[i][j]);
                j-=1;
                n+=1;
            }jO+=1;j+=1;i-=1;if (n==nmax) return ans;
            while(i>=iO){
                ans.push_back(matrix[i][j]);
                i-=1;
                n+=1;
            }iO+=1;i+=1;j+=1;if (n==nmax) return ans;
        }
        return ans;
    }
};
```
