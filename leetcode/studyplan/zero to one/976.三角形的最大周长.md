# LeetCode 976 - 三角形的最大周长
`https://leetcode.cn/problems/largest-perimeter-triangle/description/?envType=study-plan-v2&envId=programming-skills`

给定由一些正数（代表长度）组成的数组 nums ，返回 由其中三个长度组成的、面积不为零的三角形的最大周长 。如果不能形成任何面积不为零的三角形，返回 0。

## 思路

先对数组进行sort，再看这个数组最大的三个数能不能组成三角形，如果不能就将窗口向下移动，直到找到后返回周长或者找不到返回0。

## 标签

- 贪心算法
- 排序
- 数组

## 初始代码

```cpp
class Solution {
public:
    int largestPerimeter(vector<int>& nums) {
        sort(nums.begin(),nums.end());
        int szn=nums.size();
        int max=nums[szn-1];
        int mid=nums[szn-2];
        int less=nums[szn-3];
        if (mid+less>max) return less+mid+max;
        for (int i=szn-4;i>=0;i--){
            max=mid;mid=less;less=nums[i];
            if (mid+less>max) return less+mid+max;
        }
        return 0;
    }
};
```
