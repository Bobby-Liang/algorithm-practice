# LeetCode 1502 - 判断能否形成等差数列
`https://leetcode.cn/problems/can-make-arithmetic-progression-from-sequence/?envType=study-plan-v2&envId=programming-skills`

给定一个数组，如果原数组或对数组进行重新排列，可以形成一个等差数列，那么则返回true，否则返回false

## 思路

先排列再分别计算差值

## 标签

- 排序
- 数组

## 初始代码

```cpp
class Solution {
public:
    bool canMakeArithmeticProgression(vector<int>& arr) {
        sort(arr.begin(),arr.end());
        int sza=arr.size();
        for(int i=0;i<(sza-2);i++){
            if ((arr[i] - arr[i+1])!=(arr[i+1]-arr[i+2])){
                return false;
            }
        }
        return true;
    }
};
```
