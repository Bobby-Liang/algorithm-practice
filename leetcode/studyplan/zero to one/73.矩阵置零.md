`https://leetcode.cn/problems/set-matrix-zeroes/description/?envType=study-plan-v2&envId=programming-skills`

给定一个 m x n 的矩阵，如果一个元素为 0 ，则将其所在行和列的所有元素都设为 0 。请使用 `原地算法` 。

## 思路

a.遍历数组，把是0的地方的坐标记录到vector<pair<int,int>>里，最后再遍历这个新创建的数组，把对应的每一行和列的所有元素都变成0。
b.不使用vector而是转向使用两个set，这样相同的行和列就不会重复计算，重复把元素变为0了，降低了复杂度。

## 初始代码

### a

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int szi=matrix.size();int szj=matrix[0].size();vector<pair<int,int>> vp;
        for(int i=0;i<szi;i++){
            for(int j=0;j<szj;j++){
                if(matrix[i][j]==0) vp.push_back({i,j});
            }
        }
        for (pair p:vp){
            for (int i=0;i<szi;i++){
                matrix[i][p.second]=0;
            }
            for (int j=0;j<szj;j++){
                matrix[p.first][j]=0;
            }
        }
    }
};
```
### 问题

- 1.使用了额外的一个vector，空间复杂度和时间复杂度都偏高。
- 2.并非题目要求的原地算法。

### 时/空间复杂度

- 时间复杂度：O(mn(m+n))
- 空间复杂度：O(mn)

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int szi=matrix.size();int szj=matrix[0].size();set<int> row;set<int> col;
        for(int i=0;i<szi;i++){
            for(int j=0;j<szj;j++){
                if(matrix[i][j]==0){
                    row.insert(i);
                    col.insert(j);
                }
            }
        }
        for(int i:row){
            for(int j=0;j<szj;j++){
                matrix[i][j]=0;
            }
        }
        for(int j:col){
            for(int i=0;i<szi;i++){
                matrix[i][j]=0;
            }
        }
    }
};
```

## 最终代码

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int szi=matrix.size();int szj=matrix[0].size();bool row=false;bool col=false;
        if (matrix[0][0]==0){
            row=true;col=true;
        }
        if (!row){
            for (int i=0;i<szi;i++){
                if (matrix[i][0]==0){
                    row=true;
                    break;
                }
            }
        }
        if (!col){
            for (int j=0;j<szj;j++){
                if (matrix[0][j]==0){
                    col=true;
                    break;
                }
            }
        }
        for(int i=1;i<szi;i++){
            for(int j=1;j<szj;j++){
                if(matrix[i][j]==0){
                    matrix[0][j]=0;
                    matrix[i][0]=0;
                }
            }
        }
        for (int i=1;i<szi;i++){
            if (matrix[i][0]==0){
                for (int j=0;j<szj;j++) matrix[i][j]=0;
            }
        }
        for (int j=1;j<szj;j++){
            if (matrix[0][j]==0){
                for (int i=0;i<szi;i++) matrix[i][j]=0;
            }
        }
        if (row){
            for (int i=0;i<szi;i++) matrix[i][0]=0;
        }
        if (col){
            for (int j=0;j<szj;j++) matrix[0][j]=0;
        }
    }
};
```

### 改进
- 1.仅额外开辟了两个int变量和bool变量，空间复杂度低。
- 2.符合 原地算法 要求

### 时/空间复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
