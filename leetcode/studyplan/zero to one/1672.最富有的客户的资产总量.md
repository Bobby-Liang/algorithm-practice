# LeetCode 1672 - 最富有的客户的资产总量
`https://leetcode.cn/problems/richest-customer-wealth/description/?envType=study-plan-v2&envId=programming-skills`

给你一个 m x n 的整数网格 accounts ，其中 accounts[i][j] 是第 i​​​​​​​​​​​​ 位客户在第 j 家银行托管的资产数量。返回最富有客户所拥有的 资产总量 。
客户的 资产总量 就是他们在各家银行托管的资产数量之和。最富有客户就是 资产总量 最大的客户。

## 思路

对每个数组分别求和，再进行比较，输出最大的。

## 标签

- 矩阵
- 数组

## 初始代码

```cpp
class Solution {
public:
    int maximumWealth(vector<vector<int>>& accounts) {
        int a=0;int b=0;
        for(vector<int> v:accounts){
            a=accumulate(v.begin(),v.end(),0);
            if (a>b) b=a;
        }
        return b;
    }
};
```

## 最终代码

```cpp
class Solution {
public:
    int maximumWealth(vector<vector<int>>& accounts) {
        int a=0;int b=0;
        for(const vector<int>& v:accounts){
            a=accumulate(v.begin(),v.end(),0);
            if (a>b) b=a;
        }
        return b;
    }
};
```

- 1.在遍历的时候不对原数组进行复制，直接调用。

- 空间复杂度：O(1)
