# LeetCode 657 - 机器人能否返回原点
`地址`

机器人从（0，0）开始上下左右移动，每次移动给位置+-1，给定一个字符串表示移动过程，U，D表示上下，R，L表示左右。判断移动完之后是否返回（0，0），是返回true，否返回false。

## 思路

由于输入确定，量少，则不使用哈希表，先计算size，如果是奇数直接返回false，否则继续。设定两个int变量，对int1，U+1 D-1。对int2，R+1 L-1。遍历完后判断int1和2是否都为0，是返回true，反之亦然。

## 初始代码

```cpp
class Solution {
public:
    bool judgeCircle(string moves) {
        int sz = moves.size();
        int UAD=0;int RAL=0;
        if (sz%2 != 0) return false;
        else {
            for (char c:moves){
                if (c=='U') UAD+=1;
                else if (c=='D') UAD-=1;
                else if (c=='R') RAL+=1;
                else if (c=='L') RAL-=1;
            }
        }
        return (UAD==0&&RAL==0);
    }
};
```
