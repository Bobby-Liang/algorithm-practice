# 计算机与游戏开发成长节奏规划

> 目标概括：
>
> 1. 提升算法与编程能力（LeetCode + C++）
> 2. 提升游戏开发能力（Unreal Engine）

> 5. 为未来实习和就业打基础

---

# 一、核心原则

你当前最重要的任务不是“什么都学一点”，而是建立一个稳定、可持续的成长系统。

## 优先级排序

1. **算法与数据结构** —— 为实习和技术面试打基础
2. **UE 项目开发** —— 为作品集和比赛打基础
3. **Game Jam** —— 训练短周期交付能力
4. **计算机基础** —— C++、Git、操作系统等
5. **课程任务** —— 保持正常成绩即可

# 三、最推荐的日常节奏（学期版）

## 上午：算法训练

### 1~2 道 LeetCode

每道题按以下流程：

1. 阅读题目
2. 不看答案独立思考
3. 手写思路
4. 编码实现
5. 调试
6. 整理 Markdown 笔记
7. 总结收获

### 目标

- 熟练掌握常见数据结构
- 提升代码能力
- 为面试打基础

---

## 下午：课程与零碎事务

- 上课
- 作业
- 使用课堂时间思考算法思路
- 阅读技术文章

---

## 晚上：UE 项目开发

集中做以下事情：

- 蓝图 / C++ 编写
- 美术资源整合
- Bug 修复
- 打包测试
- GitHub 提交

### 每晚目标

至少完成以下之一：

- 新增一个功能
- 修复一个核心 Bug
- 优化一个系统
- 整理一次项目文档
